# Laravel Jenny House

This project resulted from a case study to increase sales using a website. Using PHP - Laravel | MySQL, a website that can manage basic operations such as displaying the most purchased products, viewing product details, adding products and viewing them in the cart.

This is my initial learning, you can even try it without logging in. Or you can log in, so that your address data can be saved. Features that might be used with JS, cannot be used yet, such as searchers, filters, adding product quantities, etc.

This repository is suitable for early learning to build an online shopping website.
